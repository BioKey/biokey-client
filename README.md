## Installation
Requires you to install [maven](https://maven.apache.org/install.html) as your build tool.

## Testing
`mvn test`

## Documentation
To generate documentation, execute:
`./document.sh`
Documentation can be found on [github pages](https://biokey.github.io/biokey-client/).
